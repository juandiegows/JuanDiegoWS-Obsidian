# 01 — Planeación y Dirección de Videojuegos Indies

---

## 🎮 ¿Qué es un Videojuego Indie?

Un videojuego independiente (indie) es desarrollado por individuos o equipos pequeños sin el apoyo financiero de una gran distribuidora.

**Ventajas:**
- Total libertad creativa
- Equipos pequeños y ágiles
- Menor presupuesto requerido
- Nichos de mercado específicos

**Desventajas:**
- Marketing limitado
- Menor alcance inicial
- Recursos limitados

---

## 📋 El Game Design Document (GDD)

El GDD es el blueprint de tu videojuego. Debe incluir:

| Sección | Contenido |
|---|---|
| **Concepto** | Premisa, género, plataforma objetivo |
| **Mecánicas** | Cómo se juega, controles, sistemas |
| **Historia** | Narrativa, personajes, mundo |
| **Arte** | Estilo visual, referencias, paleta de colores |
| **Sonido** | Ambiente, música, efectos |
| **Nivel de diseño** | Estructura de niveles, progresión |
| **Monetización** | Modelo de negocio (F2P, premium, DLC) |
| **Cronograma** | Milestones y fechas |

---

## 🔄 El Ciclo de Desarrollo

```
Concepto → Prototipo → Alpha → Beta → Gold → Lanzamiento
    ↑                                              ↓
    ←←←←← Iteración y feedback del usuario ←←←←←←
```

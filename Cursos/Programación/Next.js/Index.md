## 🔑 ¿Por qué Next.js?

| Feature | Next.js | React puro |
|---|---|---|
| SSR (Server-Side Rendering) | ✅ Nativo | ❌ Manual |
| SSG (Static Generation) | ✅ Nativo | ❌ Manual |
| Routing basado en archivos | ✅ Automático | ❌ Manual (React Router) |
| API Routes | ✅ Incluido | ❌ Servidor separado |
| Optimización de imágenes | ✅ next/image | ❌ Manual |
| SEO | ✅ Mucho mejor | ⚠️ Limitado en CSR |

## 🔑 Conceptos Core de React

| Concepto | Descripción |
|---|---|
| **Component** | Bloque reutilizable de UI (función o clase) |
| **Props** | Datos que pasan de padre a hijo (solo lectura) |
| **State** | Datos internos del componente (reactivos) |
| **Hooks** | Funciones especiales (useState, useEffect, etc.) |
| **Virtual DOM** | Representación en memoria del DOM real |
| **JSX** | Sintaxis que combina JS y HTML |

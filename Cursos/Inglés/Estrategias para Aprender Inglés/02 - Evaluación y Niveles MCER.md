# 02 — Evaluación y Niveles MCER

---

## 📊 El Marco MCER (CEFR)

El **Marco Común Europeo de Referencia** es la escala global para medir habilidades en lectura, escritura, escucha y habla.

| Nivel | Categoría | Descripción |
|---|---|---|
| **A1** | Acceso | *"My name is Ana"* — Expresiones muy simples |
| **A2** | Plataforma | Tareas simples y rutinas diarias |
| **B1** | Intermedio | Entender puntos principales, escribir textos sencillos |
| **B2** | Intermedio Alto | Fluidez, ideas complejas |
| **C1** | Dominio | Contenido exigente, estructura y matices |
| **C2** | Maestría | Precisión total, nivel nativo |

---

## 🎯 Herramientas de Medición

### Placement Test (Test de Ubicación)
- **Objetivo:** Estimar tu nivel actual rápidamente
- **Resultado:** Ruta de aprendizaje personalizada
- **Duración:** 20–30 minutos
- ⚠️ No es una certificación oficial

### Examen Oficial
- **Objetivo:** Prueba certificada para trabajo, visa o universidad
- **Duración:** Varias horas
- ✅ Certificado válido internacionalmente

## 📝 Exámenes Oficiales — ¿Cuál Elegir?

| Examen | Puntaje | Ideal Para |
|---|---|---|
| **IELTS** | 0 a 9 | Migración y educación global |
| **TOEFL iBT** | 0 a 120 | Universidades (especialmente EE.UU.) |
| **Cambridge** | Por nivel (A2–C2) | Resultados que **no caducan** |
